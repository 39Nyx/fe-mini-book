# 学习指南

本站系统整理了前端面试的核心知识点，按照以下模块进行组织：

## 📚 内容模块

| 模块 | 说明 | 链接 |
|------|------|------|
| **JavaScript 核心** | 原型链、闭包、事件循环、异步编程、ES6+ | [查看](/guide/javascript) |
| **Vue 源码解析** | 响应式原理、虚拟 DOM、编译优化 | [查看](/guide/vue) |
| **React 原理** | Fiber 架构、Hooks、状态管理 | [查看](/guide/react) |
| **工程化实践** | Webpack/Vite、Babel、CI/CD | [查看](/guide/engineering) |
| **浏览器与网络** | 渲染原理、HTTP、缓存、性能优化 | [查看](/guide/browser) |
| **数据结构与算法** | 经典数据结构、算法题解析 | [查看](/guide/algorithm) |
| **手写代码专题** | Promise、防抖节流、深拷贝等 | [查看](/guide/handwriting) |
| **场景题与设计模式** | 业务场景、设计模式、架构设计 | [查看](/guide/scenario) |

## 🎯 使用建议

1. **系统学习**：按照模块顺序循序渐进
2. **重点突破**：针对薄弱知识点专项复习
3. **手写练习**：代码题一定要动手实践
4. **模拟面试**：结合场景题进行自测
