## 学习建议

1. **理解原理**：不只是背诵代码，要理解为什么这样写
2. **边写边测**：在控制台或编辑器中实际运行代码
3. **对比标准**：手写实现后与原生 API 对比差异
4. **举一反三**：掌握核心思想后，能手写类似功能

## 高频考点总结

- Promise 原理及静态方法
- this 绑定（call/apply/bind）
- 函数柯里化与反柯里化
- 防抖节流（注意 immediate 参数）
- 数组方法（注意稀疏数组处理）
- 深拷贝（处理循环引用和特殊类型）
- 继承（寄生组合继承为最优方案）
- 发布订阅模式
