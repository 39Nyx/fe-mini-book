## 面试考点速览

### 场景题常见类型

1. **组件设计**：Modal、Toast、Tooltip、Dropdown 等基础组件
2. **性能优化**：虚拟列表、懒加载、防抖节流应用
3. **交互实现**：拖拽排序、无限滚动、富文本编辑器
4. **文件处理**：大文件上传、断点续传、图片压缩
5. **复杂业务**：购物车、权限系统、表单引擎

### 设计模式速查

| 模式 | 用途 | 前端应用 |
|------|------|---------|
| 单例模式 | 全局唯一实例 | Store、配置管理 |
| 工厂模式 | 对象创建封装 | 组件创建、UI 适配 |
| 观察者模式 | 状态变更通知 | Vue 响应式系统 |
| 发布订阅模式 | 解耦事件通信 | Event Bus、消息中心 |
| 策略模式 | 算法互换 | 表单校验、支付渠道 |
| 装饰器模式 | 动态扩展功能 | AOP、方法增强 |
| 代理模式 | 访问控制 | 图片预加载、数据代理 |

## 高频面试题

### 1. 观察者模式和发布订阅模式的区别？

**答案**：
- **观察者模式**：Subject 和 Observer 直接通信，耦合度较高
- **发布订阅模式**：通过事件中心通信，发布者和订阅者完全解耦

### 2. 如何实现一个前端路由？

**核心要点**：
- **Hash 模式**：监听 `hashchange` 事件，兼容性好
- **History 模式**：使用 `pushState`/`replaceState`，需要服务端配合
- **内存路由**：适用于非浏览器环境（如测试、移动端）

### 3. 如何设计一个组件库？

1. **目录结构**：清晰分离组件、样式、文档、测试
2. **样式方案**：CSS-in-JS、CSS Modules、BEM
3. **构建输出**：ESM、CJS、UMD 多格式
4. **类型支持**：TypeScript 定义文件
5. **主题系统**：CSS 变量、配置化
6. **文档站点**：Storybook、dumi
7. **测试覆盖**：单元测试、E2E 测试
